# Simple XOA Community Updater
A simple Xen Orchestra Community Edition updater

# Prerequisites
Xen Orchestra must be installed from sources using [this guide](https://xen-orchestra.com/docs/from_the_sources.html) from Xen Orchestra or using a script like [xenorchestra_installer](https://github.com/Jarli01/xenorchestra_installer).

# Usage
Run [update_xoa.sh](./update_xoa.sh) with root permissions
